Remote Job Search

A job board where all jobs are remote

If you're interested only in remote jobs, this site is for you. It utilizes an API call connected to Remotive that compiles all the remote jobs. Simply type in the keyword in the search bar it will show the results below. If any of the jobs look interesting, you can then add them to your favorites, which are dispayed on the right. You can then click for more details and a pop up will present full job description with a link to apply. Built on React, it uses mainly react-bootstrap for styling and react-reveal for animation effect.
